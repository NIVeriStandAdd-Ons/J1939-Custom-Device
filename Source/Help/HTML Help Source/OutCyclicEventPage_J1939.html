J1939 Outgoing Cyclic Event Triggered Page

J1939 Outgoing Cyclic Event Triggered contains any cyclic event J1939 frames selected to generate.

What do you want to do?

Import frames

To import frames, select Event Triggered in the configuration tree and click the Import Frames button. You also can right-click Single-Point in the configuration tree and select Import Frames from the shortcut menu. Note that you can just select the J1939 parameters of interest within a given frame. You do not have to select the entire frame.

Delete multiple frames

To delete multiple frames, right-click Event Triggered in the configuration tree and select Delete Multiple Frames from the shortcut menu.
