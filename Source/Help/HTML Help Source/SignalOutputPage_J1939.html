J1939 Parameter Page

Parameter is a J1939 parameter previously imported.

What do you want to do?

Select Counter or Protection Signal

To select the counter or protection signal, right-click the desired signal in the configuration tree and select Make Counter Signal or Make Protection Signal.

Delete

To delete a signal, right-click the desired signal in the configuration tree and select Delete from the shortcut menu.

Set default value

To set a default value for the signal, enter the desired value in the Default Value field.
